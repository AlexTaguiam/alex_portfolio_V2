Web App Completed

Sari-Sari Store Tracker

A web application built to help sari-sari store owners manage customer credit (utang) efficiently. Tracks who owes what, logs transaction history, and provides a simple dashboard so store owners can stop relying on paper records and manual tallying.

  • Customer credit (utang) tracking & management
  • Transaction history & running balance logs
  • Simple dashboard for quick overviews
  • Search & filter customers by name or balance
MongoDB Express.js React Node.js

Desktop App Completed

Full Bus Reservation System

A full-featured desktop bus reservation system developed with VB.NET and MySQL. Passengers can browse available routes and seats, make bookings, and receive confirmation. Admins manage schedules, buses, and passenger records through a dedicated control panel.

  • Seat selection & real-time availability display
  • Booking management with passenger records
  • Admin panel for routes, buses & scheduling
  • MySQL-backed data storage & reports
VB.NET MySQL Windows Forms

Full-Stack Web In Progress

Parlor Reservation System

A PERN-stack web application for a salon & gown rental parlor. Customers can browse available services, book appointments, and reserve gowns or suits for events. Built end-to-end with PostgreSQL, Express, React, and Node.js for a modern, responsive experience.

  • Online booking for salon & beauty services
  • Gown & suit inventory browse and reservation
  • Calendar-based availability & slot management
  • Customer profiles & booking history
PostgreSQL Express.js React Node.js PERN Stack
